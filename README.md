# Vandy Academic Planner

## Starting Development

Open three terminals:

**Terminal 1 - Database:**
```bash
supabase start
```

**Terminal 2 - Backend:**
```bash
cd backend
npm run dev
```

**Terminal 3 - Frontend:**
```bash
cd frontend
npm run dev
```

## Stopping Development

Press `Ctrl+C` in backend and frontend terminals, then:

```bash
supabase stop
```
